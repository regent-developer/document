# MCP
MCP（Model Context Protocol，模型上下文协议）由Anthropic于2024年底开源，其核心是通过标准化接口实现大模型与外部工具的“即插即用”。类比互联网的TCP/IP协议，MCP构建了AI交互的通用语言。

## MCP Server支持两种主要连接模式：
* Stdio模式：通过命令行进程通信，适合本地开发
* SSE模式：通过HTTP连接，适合生产环境部署
