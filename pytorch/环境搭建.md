# PyTorch 环境搭建

前提：安装 Anaconda（64 位版）

## 安装

- 创建 pytorch 的环境

```shell
conda create -n pytorch python=3.7
```

- 切换环境

```shell
conda activate pytorch
```

- 安装 PyTorch（该方法安装失败）

1. 在官网（https://pytorch.org/get-started/locally/）中选择相应系统等信息后，生成安装命令
2. 执行安装命令

```shell
conda install pytorch torchvision torchaudio cpuonly -c pytorch
```

- 安装 PyTorch

1. 在清华镜像下载 pytorch 和 torchvision
   https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/win-64/
2. 执行离线安装命令

```shell
conda install --offline torchvision-0.4.0-py37_cu100.tar.bz2
conda install --offline pytorch-1.0.0-py3.7_cuda100_cudnn7_1.tar.bz2
```

## 验证是否安装成功

- 在 pytorch 环境中输入命令 python
- 之后输入 import pytorch，如不报错即安装成功
- 输入 torch.cuda.is_available，true：支持 GPU，否则只支持 CPU
