# electron
Electron 可以让你使用纯 JavaScript 调用丰富的原生 APIs 来创造桌面应用。

## 环境搭建

### 安装Node.js和npm

### 安装cnpm
```
npm install -g cnpm --registry=https://registry.npm.taobao.org

cnpm -v
```

### 安装Electron
```
npm install -g electron
```

```
cnpm install -g electron
```

安装是否成功
```
electron --version
```

### 运行
```
npm start

electron .
```
